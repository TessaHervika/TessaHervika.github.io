Pertemuan 1

    Pada pertemuan 1 Mata Kuliah Rekayasa Web belajar Cara install, backup, dan restore Wordpress

    Cara Install Wordpress

  1. 1. Download file zip wordpress di halaman wordpress.org
  2. 2. Ekstrak file zip tersebut
  3. 3. Rename nama folder (usahakan nama folder jangan ada spasi, contoh: si5a2122o)
  4. 4. Xampp control harus dalam keadaan learning
  5. 5. Pindah file ke folder xampp -> htdocs
  6. 6. Ingat nama folder saat akan di akses. Contoh (localhost/si5a2122o)
  7. 7. Setelah di akses pilih bahasa
  8. 8. Siapkan database dengan membuka xampp control lalu klik admin pada MySQL
  9. 9. Akan muncul halaman localhost/phpmyadmin/
  10. 10. Isi keterangan koneksi basis data (pastikan databasenya sudah ada di phpMyAdmin kita)
  11. 11. Klik kirim & Jalankan Sekarang
  12. 12. Masukkan Informasi yang dibutuhkan
  13. 13. Klik Instal Wordpress
  14. 14. Setelah berhasil coba login
  15. 15. Web pun sudah jadi

    16. Cara Backup Wordpress

  16. 1. Buat satu buah folder baru
  17. 2. Copy folder yangg sudah di rename saat instal tadi (contoh: si5a2122o)
  18. 3. Paste di folder yang baru
  19. 4. Lalu dari phpMyAdmin klik export, pilih custom dan pilih database yang telah kita buat tadi
  20. 5. Klik tombol Go
  21. 6. Simpan di folder baru tadi
  22. 7. Backup selesai
  23. 8. Compress file-file tersebut menjadi satu folder

    24. Cara Restore Wordpress

  24. 1. Ekstrak folder yang telah di backup
  25. 2. Lalu import backup database
  26. 3. Pilih file database
  27. 4. Klik Go
  28. 5. Jika sudah berwarna hijau, restore pun berhasil
